# Casual Minds Matter CIC — Claude Code Context

## Overview
Casual Minds Matter CIC — Mental health, wellbeing and housing support in Burnley. Community Interest Company website.

**Stack**: HTML | CSS | JavaScript
**Hosting**: GitHub Pages
**Branch**: main

## Key Patterns
- Static informational site for CIC
- Service descriptions and support information
- Contact forms and referral pathways
- Accessibility-first design (mental health audience)

## Rules
- British English throughout
- Sensitive, supportive tone — this is a mental health service
- WCAG AA accessibility compliance minimum
- Clear navigation to crisis support resources
- Mobile-first responsive design
- Fast loading — users may be on limited data
- Include helpline numbers prominently
- Privacy-conscious — no unnecessary tracking
